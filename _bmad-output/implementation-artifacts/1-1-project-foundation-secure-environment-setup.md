I want Mongoose dependencies and a secure environment template,

   **Then** `require('dotenv').config()` is called at process entry before any `MONGODB_URI` reads


Optional facilitator comments (e.g. `# Paste facilitator-provided Atlas URI`) are acceptable but must not include real credentials.

**`server.js` dotenv placement (exact pattern):**

```javascript
require('dotenv').config();

const express = require('express');
// ... rest unchanged for this story
```

Place before all other `require()` calls so `process.env.PORT` and future `process.env.MONGODB_URI` reads work when `.env` exists.
